Cross-device sync

Paste a GitHub personal access token with only the gist scope to sync your start-time schedule across devices via a private Gist in your own GitHub account. The token is stored only in this browser (in localStorage) — it is never written into this app's code or repo. Create one at github.com/settings/tokens. Every device needs this same step — paste the same token and tap Connect on each phone/laptop/tablet you want kept in sync; a device that's never connected has no way to know about changes made elsewhere.
